# Roster Proc scripts

## Setup and bootstrap

```
brew install pyenv
```

```
brew update && brew upgrade pyenv
pyenv install 3.7.6
```

```
pip install -r requirements.txt
```

Add to your ~/.bash_profile or ~/.zshrc

```
# Load pyenv on shell
eval "$(pyenv init -)"%
```

More information [here](https://github.com/pyenv/pyenv).
